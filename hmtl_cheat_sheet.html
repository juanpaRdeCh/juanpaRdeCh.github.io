Este tag funciona para escribir encabezados tal y como el que tenemos arriba de este parrafo. Esta etiqueta se puede escribir de h1 a h6 y la difetencia entre cada uno de ellos sera el tamaño de la fuente. Ejemplo:

div - Contenedor en bloque

Esta etiqueta sirve para poder crear una division dentro de un parrafo para de esta manera añadir estilos a nuestro texto. Lo importante de esta etiqueta es que funciona en forma de bloque, por lo que se podran agrupar multiples parrafos o encabezados Por ejemplo:

Este parrado de texto se encuentra en color azul en lugar del negro por defecto

Este segundo parrrafo se encuentra agrupado dentro de la misma division del primero por lo que sigue estando escrito en color azul

span - Contenedor en Linea

Estea etiqueta crea una division al igual que la pasada, sin embargo esta funciona como un contenedor en linea en lugar de uno en bloque como el caso anterior. Ejemplo:

Este texto solo tiene la palabra azul en color azul en lugar de haber cambiado el formato del parrafo entero.

aside - aparte

Como su nombre lo sugiere, esta etiqueta sirve para crear una division semantica en la cual se incorpore informacion indirectamente relacionada al resto del texto. Ejemplo:

La clase de construccion de software y toma de decisiones nos enseña aspectos basicos y avanzados de la programacion

La programacion requiere de una estructura semantica precisa para funcionar

footer - Pie de pagina

El footer funciona como un pie de pagina, en este se pueden anexar referencias, etc. Este elemento funciona en bloque. Ejemplo:

Referenicas: Dr. Octavio Navarro Hinojosa | Link Canvas |

nav - navegacion

La intencion de nav es poder crear un bloque de links de navegacion para tenerlos agrupados en un mismo lugar. Sin embargo, no es necesario que todos los links esten aqui presentes. Ejemplo:

Entra bajo tu propio riesgo | Entra bajo tu propio riego 2

hr - cambio de tema

Esta etiqueta sirve para crear una linea horizontal a lo ancho de la pagina. Esta linea representa un cambio de tema dentro del documento. Ejemplo:

Artistas como Giorgio Moroder, Daft Punk o Tyler the Creator han ganado mucha popularidad en los ultimos años debido a que han roto el concepto general que se tenia sobre la musica, creando estilos unicos y diferentes.

El cine es una forma de arte sujeta a la interpretacion del publico, haciendola de esta manera una forma de expresion.

table - tabla

El tag "table" crea una tabla dentro del documento html. Esta tabla consiste de tr, th y td, asociados a los headers, columnas y filas. Ejemplo:

ul - unordered list

Este tag funciona para crear una lista dentro de un documento html. Este tiene que ser usado con el tag "li" para crear cada viñeta. Ejemplo:

Lista del super

  • Crema de cacahuate: 800g
  • Pollo: 1.5kg
  • Arrachera de res: 1.5kg
  • Brocoli: 1 bolsa
  • Espinaca: 1 bolsa
